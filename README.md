## üìñ Overview

This project is a creative take on a personal portfolio website, designed to mimic the **Kali Linux** operating system. It provides an immersive "hacker" experience that starts from the boot sequence, goes through a realistic login screen, and lands on a functional desktop environment where users can explore "apps" to learn about my skills, projects, and background.

## üöÄ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Kali-Portfolio.git
   cd Kali-Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
